## Prerequisites

* JDK 1.8
* JRE 1.8
* Android Studio 2.3

## Development

1. Clone repository locally
2. Open cloned repository with Android Studio

* Before committing, ensure the build is not broken and linted

```
bash ./gradlew assemble
bash ./gradlew lint --stacktrace
bash ./gradlew check
```
